# the Hofmann Elimination (usage)

## Design

This module provides the common integration files needed for efficient usage
of the other two modules included in this project;  `oprf` and `opaque`. Those
modules provides the fundamental security primitives. With the goal of
reducing the attack surface commonly seen in today's applications. The 
hofmann module itself implements the common elements needed by today's tooling,
such as dropwizard and spring boot.

The overall format for the packages listed here is following the manager-model
pattern as shown below:
